My Projects

Monthly Rent Splitter 

A simple web application that calculates and splits monthly rent and shared household expenses among multiple people. Users can input costs such as rent, utilities, groceries, and maintenance, and the app instantly calculates each person's share. The application features real-time calculations, responsive design for mobile and desktop, and a dark/light mode toggle for better user experience.

Technologies: HTML, CSS, JavaScript.

Bubble Hit Game 

A fun and interactive web game where players click on randomly appearing bubbles to earn points. The game features random bubble generation, real-time score tracking, and a simple, intuitive interface that runs directly in any web browser—no installation required.

Technologies: HTML, CSS, JavaScript.

Dynamic Quiz App

An interactive web-based quiz application that fetches questions from the Open Trivia Database (OpenTDB) API. Users can select categories, answer multiple-choice questions within a timer, and see their score updated in real-time. The app features responsive design, making it mobile-friendly and easy to use on any device.

Technologies: HTML, CSS, JavaScript, OpenTDB API.

Sidcup Family Golf Website Clone 

This project involved creating a clone of the Sidcup Family Golf website to replicate its visual design and interactive elements. Features include scroll-based and timeline animations, dynamic user-triggered effects, and smooth, responsive transitions, combining GSAP with CSS and JavaScript to create an engaging user experience.

Technologies: HTML, CSS, JavaScript, GSAP.

Mentor Me 

Developed a web-based Mentor Management System to digitally manage mentor and mentee information. The system allows administrators to add, update, and track mentor–mentee relationships through a centralized platform. It improves data organization and reduces manual record keeping by storing information in a MySQL database and enabling Excel-based data entry and reporting for efficient bulk management.

Technologies Used: HTML, CSS, JavaScript, PHP, MySQL, Excel.
